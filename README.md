# Overview

[paepche.de/reportstyle](https://paepcke.de/reportstyle)

Internal Package for uniform reporting across \*info packages. 

- enables uniform data reports look & feel for:
	- plaintext
	- ansi color terminal 
	- markdown 
	- html

- supported packages:
	- paepcke.de/dnsinfo
	- paepcke.de/certinfo
	- paepcke.de/tlsinfo
	- ...
